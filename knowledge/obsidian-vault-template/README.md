# HESOYAM Obsidian Vault Template

A PARA-method vault structure pre-configured for Claude Code integration.

## Quick Start

1. Copy this folder as your new Obsidian vault
2. Open it in Obsidian
3. Install the Terminal plugin (for Claude Code access)
4. Install Obsidian Skills: `/plugin marketplace add kepano/obsidian-skills`
5. Run Claude Code inside the vault: `claude`

## Structure

```
00_Inbox/        → Quick capture. Process daily.
01_Projects/     → Active, time-bound work. One folder per project.
02_Areas/        → Ongoing responsibilities. Architecture, team, etc.
03_Resources/    → Reference material. Patterns, decisions, debug journal.
04_Archive/      → Completed work. Don't delete, archive.
05_Attachments/  → Images, PDFs, files.
06_Templates/    → Note templates for consistency.
.claude/skills/  → Agent skills for Claude Code.
```

## Templates Included

- `decision-record.md` — Architecture Decision Record
- `debug-journal.md` — Bug investigation and resolution log
- `daily-note.md` — Daily standup and work log
- `meeting-notes.md` — Structured meeting capture

## Customize

This is a starting point. Adapt the structure to how you think, not how someone told you to think.
